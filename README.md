# AlphaAI 🤖

A ChatGPT-like AI chat application built with React and Node.js.

## 🚀 Live Demo
alpha-ai-6u88.vercel.app

## 🛠️ Tech Stack
- **Frontend:** React, Vite, CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **AI:** OpenAI API

## ✨ Features
- Real-time AI responses with typing animation
- Chat history saved in sidebar
- Markdown & code highlighting support
- New chat functionality

## ⚙️ Setup
1. Clone the repo
2. Add `.env` in Backend with `OPENAI_API_KEY` and `MONGODB_URI`
3. Run `npm install` in both Frontend and Backend
4. Run `npm start` in Backend and `npm run dev` in Frontend
